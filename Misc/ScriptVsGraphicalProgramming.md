# Script Programmierung vs. Grafische Programmierung

## Vorteile Script Programmierung
* Skalierbarkeit
* Performance
* Wiederverwendbarkeit
* Programmier-Aufwand viel geringer

## Vorteile Grafische Programierung (mit Flussdiagramm etc.)
* Fehlertoleranz ist höher
* Prototyping ist einfacher (?)
* Komplexität ist geringer (?)
